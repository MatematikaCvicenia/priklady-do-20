# Matematické príklady do 20 pre prvákov

Jednoduchá interaktívna webová stránka na precvičovanie sčítania a odčítania do 20.

## Funkcie
- Dve kategórie úloh:
  - príklady s neznámou `x` (napr. x + 7 = 12)
  - klasické príklady bez x (napr. 8 + 9 = )
- Možnosť vybrať počet čísel (2–5)
- Možnosť vybrať len sčítanie / len odčítanie / mix
- Kreslenie na pomocnú tabuľku myšou
- Vyhodnotenie správnosti
